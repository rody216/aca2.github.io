Glosario de términos HTML


  1. HTML: HyperText Markup Language. Es el lenguaje de marcado utilizado para crear páginas web.

  2. Etiqueta: Un elemento de HTML que se utiliza para estructurar y dar formato al contenido de una página web.

  3. Atributo: Un valor adicional que se puede agregar a una etiqueta HTML para modificar su comportamiento o apariencia.

  4. Enlace: Un elemento de HTML que permite a los usuarios navegar entre diferentes páginas web o secciones de una misma página.

  5. Imagen: Un elemento de HTML que se utiliza para mostrar gráficos o fotografías en una página web.

  6. Listas: Un conjunto de elementos relacionados que se presentan en una página web en forma de una lista.

  7. Encabezado: Un elemento de HTML que se utiliza para marcar una sección de contenido como un encabezado.

  8. Párrafo: Un elemento de HTML que se utiliza para presentar texto en una página web.

  9. Tabla: Un elemento de HTML que se utiliza para presentar información en forma de una tabla.

  10. Formulario: Un conjunto de elementos de HTML que se utilizan para recopilar información de los usuarios en una página web.
